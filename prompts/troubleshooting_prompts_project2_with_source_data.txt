# Project 2 OCR 실습 오류 대응 프롬프트

[이미지 파일을 못 찾는 경우]
CSV의 image_filename 컬럼과 실제 data/input_images 폴더를 비교해서 누락된 경로를 찾아줘. 경로 구분자 문제인지, 파일명 대소문자 문제인지 구분해서 수정해줘.

[OCR이 한글을 잘 못 읽는 경우]
이미지 전처리를 강화해줘. grayscale, threshold, contrast enhancement, denoise를 순서대로 적용하고 원본 이미지와 전처리 이미지의 OCR 결과를 비교해줘.

[금액 숫자를 잘못 읽는 경우]
영수증 문서에서 '합계금액' 주변 영역을 우선 탐색하도록 OCR 후처리 규칙을 추가해줘. 쉼표, 원, 공백을 제거한 뒤 숫자만 파싱해줘.

[설문 점수가 누락되는 경우]
1~5점 선택 박스 주변의 체크 표시 또는 원 표시를 탐지하는 보조 규칙을 추가해줘. OCR 텍스트만으로 어려우면 이미지 좌표 기반 체크 표시 탐지를 시도해줘.

[대시보드 실행이 안 되는 경우]
필요 패키지를 확인하고 requirements.txt를 갱신해줘. Streamlit 실행 명령어와 포트를 다시 안내하고, 포트 충돌 시 다른 포트를 사용하게 해줘.
