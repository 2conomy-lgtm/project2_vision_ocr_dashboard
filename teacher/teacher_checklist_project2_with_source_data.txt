# 확인 포인트

- 이 실습은 크롤링 실습이 아니라, 제공된 원본 정형 데이터와 이미지 데이터를 이용한 OCR·정제·대시보드 실습이다.
- 수강생에게 먼저 ground_truth_multimodal_240.csv와 image_contact_sheet_preview.jpg를 열어 본다.
- 이미지 240장은 OCR 품질 평가를 위해 원본 정답 데이터와 1:1로 매칭된다.
- 수강생이 코드를 직접 이해하지 않아도 프롬프트 순서대로 Antigravity에 요청하면 된다.
- 실제 Vision API가 어려운 환경에서는 시뮬레이션 OCR 모드를 허용하되, 폴더 구조와 품질 리포트 생성은 반드시 수행한다
